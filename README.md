# Installation
  - Import SQL Patient to Database
  - Install Package.json
      ```
      nmp run i
      ```
  - Install Composer
      ```
      composer install
      ```
  - Run Webpack
      ```
      nmp run dev
      ```
  - Start your Xampp (apache and mysql)
  - Open your project
      ```
      http://localhost/newproject/
      ```
